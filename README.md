# WoD Pretty Theme

[![Foundry](https://img.shields.io/badge/dynamic/json?url=https%3A%2F%2Fgithub.com%2Fleonebeavi%2Fwod-pretty-theme%2Freleases%2Flatest%2Fdownload%2Fmodule.json&label=Foundry&query=$.compatibility.verified&colorB=fe6a1f)](https://foundryvtt.com)
[![Version](https://img.shields.io/github/v/release/leonebeavi/wod-pretty-theme?label=version&sort=semver&color=8a1320)](https://github.com/leonebeavi/wod-pretty-theme/releases/latest)

A dark, glass-gothic re-theme for the **[World of Darkness 5e](https://github.com/WoD5E-Developers/wod5e)** (`wod5e`) game system on **Foundry VTT** — restyled and restructured character/item sheets inspired by the Progeny VtM v5 character creator, with per-splat accent colors (Vampire crimson, Werewolf green, Hunter amber, Mortal slate).

> ⚠️ Unofficial, **cosmetic-only** fan module. It ships **no** wod5e code or assets and is not affiliated with the wod5e developers, Paradox Interactive or White Wolf.

## Installation

In Foundry VTT, open **Add-on Modules → Install Module**, paste the manifest URL below, click **Install**, then enable it in your world's **Manage Modules** (requires the `wod5e` system):

```
https://github.com/leonebeavi/wod-pretty-theme/releases/latest/download/module.json
```
